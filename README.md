# Project Overview

This project develops an industry-grade public safety analytics dashboard built under CoreTech Labs (10Alytics, United Kingdom) to analyze road traffic accident data and support public safety governance, infrastructure planning, and transportation risk policy modeling.

The system transforms large-scale accident records into interactive decision dashboards that reveal casualty severity patterns, infrastructure risk hotspots, vehicle-type vulnerabilities, and environmental risk factors ‚Äî supporting data-driven safety interventions and infrastructure investment prioritization.
